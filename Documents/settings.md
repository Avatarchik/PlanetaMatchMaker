# Settings

JSON format.

## Items

- enable_session_key_check: true or false
- time_out_seconds: int
- log_level: debug, info, warning, error or fatal
- ip_version: v4 or v6
- port: int
- max_connection_per_thread: int
- thread: int
- max_room_per_room_group: int
- room_group_list: a list of string
